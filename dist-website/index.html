機能

必要なものが、すべて揃っている

面倒な設定不要。拡張機能を入れてボタンを押すだけで、プロ品質のPDFが手に入ります。

ページ全体をキャプチャ

スクロールが必要な長いページも、自動で一枚のスクリーンショットに合成。見切れなし。

A4 / A3 PDF出力

印刷にそのまま使えるA4・A3サイズのPDFを生成。ヘッダーやフッターで情報を自動付与。

複数タブを一括処理

複数のページを選んでまとめてキャプチャ。1つのPDFにまとめて出力できます。

Pro

カスタムレイアウト

1〜4列の柔軟なグリッドレイアウト。資料作成や比較表示に最適なレイアウトで出力。

Pro

完全ローカル処理

スクリーンショットのデータは外部サーバーに送信しません。すべての処理がブラウザ内で完結。

取得前の強制リロード

キャプチャ直前にページをリロードするオプション。最新の状態を確実にキャプチャできます。

使い方

3ステップで完了

難しい設定は一切なし。インストールからPDF生成まで、最短1分でできます。

1

拡張機能をインストール

Chrome Web StoreからSitePrinterを追加。ボタン一つで完了します。

2

キャプチャしたいページを開く

保存したいWebページを開き、ブラウザ右上の拡張機能アイコンをクリック。

3

PDFをダウンロード

「スクリーンショットを取得」ボタンを押してプレビューを確認。そのままPDFで保存。
