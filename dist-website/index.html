FAQ

ご不明な点はお気軽にお問い合わせください。

ChromeとEdgeの両方で使えますか？

はい。Google Chrome と Microsoft Edge の両ブラウザで動作します。Chrome Web Store または Microsoft Edge Add-ons からそれぞれインストールできます。

無料プランで何ができますか？

A4サイズ・1列レイアウトでのPDF出力、複数タブの一括処理、JPEG/PNG画像形式の選択、完全ローカル処理が無料でご利用いただけます。A3サイズ・複数列レイアウト・ヘッダー/フッター付与はProプランの機能です。

キャプチャしたデータは外部に送信されますか？

いいえ。スクリーンショットのデータが外部サーバーに送信されることは一切ありません。PDF生成を含むすべての処理がお使いのブラウザ内で完結します。

ヘッダー・フッターには何が表示できますか？

ページタイトル・URL・キャプチャ日時・ページ番号をPDFに自動で付与できます。提出用資料や記録用途に便利な機能です。（Proプラン限定）

Proプランにアップグレードするには？

拡張機能のオプション画面からアカウントを作成し、アップグレードボタンよりお手続きいただけます。月額¥300で、クレジットカード決済に対応しています。

解約はいつでもできますか？

はい、いつでも解約できます。解約後は現在の請求期間の終了まで引き続きProプランをご利用いただけます。違約金や解約手数料は一切かかりません。
